Minha Biografia

Radamés Rodrigues Neto publicado em 10/07/2022

Radamés é um estudante de programação no Programa Entra 21, uma parceria da Bluesoft com a FAPESC, focada na Formação de profissionais para o ecossistema de inovação do estado de Santa Catarina. Na 17ª edição do programa, as demais regiões de Santa Catarina foram contempladas e graças a esse fato, eu pude participar.


No programa, nós temos aulas voltadas para a área de desenvolvimento web full-stack utilizando a linguagem de programação Java no back-end, além de HTML, CSS, e Javascript no front-end. Os frameworks utilizados no programa são Spring Boot e Angular.


Além do programa de formação, eu sou acadêmico da 4ª fase no curso de Análise e Desenvolvimento de Sistemas. Meus interesses em T.I. são programação web, segurança cibernética ofensiva e geopolítica do cyber-espaço.
